# edumotion

##Development

###Dependencies

You will need electron and Babel CLI to run:

    npm install -g electron
    npm install -g babel

The rest of the dependencies will be installed with

    npm install

###Compiling

    babel edumotion.js -o edumotion-compiled.js

###Running

CD to the project directory and run

    electron .
